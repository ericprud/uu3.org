LdHostManager - repo

  • type — github or... can add more
  • owner — repository group
  • name — repository name

LdHostManager - subdomain

  • creates a subdomain of the root
  • give you nice, short URLs for e.g. your FAIR Data Point (or whatever)
  • leverages hierarchical domain certificates

LdHostManager - commit

  • when — time of last commit
  • who — version control username of last commit
  • hash — ID of last commit
